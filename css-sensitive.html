CSS 本身是大小写不敏感的, 除了 CSS 之外的不可控部分. 为避免混淆, 最好统一使用小写.

CSS: 属性选择器大小写不敏感, 类选择器 和 ID选择器 和 URI 和 Font-Family 大小写敏感.

HTML: 大小写不敏感.
